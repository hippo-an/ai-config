# AI Conference (ai-conf)

## 프로젝트 목적

이 프로젝트는 Claude Code를 사용하면서 발견한 유용한 구성(configurations), 팁(tips), 그리고 베스트 프랙티스를 모아놓는 개인 지식 저장소입니다.

## 프로젝트 철학

- 필요할 때 필요한 것만 추가한다
- 미리 구조를 복잡하게 만들지 않는다
- 실제로 사용하고 검증된 내용만 기록한다
- 간결하고 실용적인 예제를 우선한다

## 작업 가이드

이 프로젝트에서 작업할 때:
- 새로운 파일/폴더는 실제 필요가 생겼을 때만 생성
- 문서는 한글로 작성 (코드/설정은 영어)
- 예제는 실제 동작하는 코드로 작성
- 각 팁/구성에는 "왜 유용한지" 설명 포함

## 현재 상태

프로젝트 초기 단계 - 필요에 따라 점진적으로 확장 중
